Project Overview

State of Mind is an interactive educational game developed for the Web Mapping course in the Cartography Master's program at the Technical University of Vienna (2025).

Centered around the theme of democracy, the game explores different political regimes around the world through engaging gameplay. Players can dive into various modes, some featuring influential historical figures.
